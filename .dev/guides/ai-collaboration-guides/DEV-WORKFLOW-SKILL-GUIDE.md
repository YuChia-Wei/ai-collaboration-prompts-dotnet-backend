# Dev Workflow 相容入口

`dev-workflow` 自 v0.6.0 起是 `software-development-orchestrator` 的
deprecated compatibility identifier，且目前沒有預排移除版本。

新工作請使用
[`SOFTWARE-DEVELOPMENT-ORCHESTRATOR-SKILL-GUIDE.md`](SOFTWARE-DEVELOPMENT-ORCHESTRATOR-SKILL-GUIDE.md)。
既有 workflow、task、assessment、release、owner skill 與 provenance
中的歷史識別保持原值。
