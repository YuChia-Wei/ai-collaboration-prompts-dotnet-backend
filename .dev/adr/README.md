# Decision Archive

本目錄已退役，不再作為日常 architecture / collaboration source of truth。

## Canonical Sources

目前專案的正式規則入口是：

- `.dev/ARCHITECTURE.MD`
- `.dev/standards/`
- `.dev/guides/`
- `AGENTS.md`

## Archive Policy

- 不新增新的舊式決策檔
- 若舊決策已完整落到 canonical docs，應直接刪除舊決策檔
- 若保留歷史檔，只能作為過渡脈絡，不可讓其他文件再依賴它

## Migration Direction

- 舊決策中的硬規則應落到 `.dev/standards/`
- 使用教學與模式說明應落到 `.dev/guides/`
- agent-facing reusable rules 應落到 `.ai/`
